# ChemAnalyzer
бета-версія програми для розраухнку вибухових характеристик вибухових сумішей(молей компонентів, брутто-формули, кисневого балансу, продуктів вибуху, теплоти вибуху)

Відкривши програму, відкриється її вікно: зверху чотири колонки з трьома стовпцями, куди вводяться компоненти та їх формули, концентрації, ентальпії утворення у кДж/моль, внизу кнопка “Порахувати”. Результат буде виводитися у нову вкладку (зверху є панель з вкладками “Введіть свої дані” та “Результат”)

Формули компонентів вводять у перший стовпчик великими латинськими літерами з індексами (наприклад, CH4). Скорочення не допускаються: етанол — C2H6O, аміачна селітра — N2H4O3. Десяткові індекси записують через крапку. У другий стовпець вводять концентрацію без знаку %, у третій — ентальпії в кДж/моль. Усі десяткові числа також записувати через крапку.
Якщо компонентів менше ніж чотири, решту колонок заповнюють формулою “H”, а числові поля — нулями, інакше програма не працюватиме. Важливий порядок заповнення компонентів: лише у четверту колону можна записувати додаткові селітри або речовини, що містять у собі Al, Ca або Na, але в неї також можна записувати інші компоненти, що не мають у своєму складі металів.
Після натискання кнопки потрібно перейти у вкладку “Результат”. Якщо дані не відображаються, слід перевірити правильність заповнення таблиці. У “Результаті” подано компоненти та їх молі, молі елементів на 1 кг суміші, кисневий баланс, молі та об’єм продуктів, теплоти вхідних речовин і продуктів, а в останньому рядку — теплоту вибуху суміші.
